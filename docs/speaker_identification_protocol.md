## SPEAKER IDENTIFICATION PROTOCOL

When attributing quotes or perspectives throughout transcripts and summaries:

1. **Use bracket notation** for all speaker names: [John Doe], [Jane Smith]

2. **Attempt identification** from context:
   - Video title (often includes speaker names)
   - Channel name/creator
   - Self-introductions in the transcript
   - Context clues (credentials, roles mentioned in discussion)

3. **Use numbered placeholders** if identity cannot be determined: [speaker01], [speaker02], [speaker03]

4. **Maintain consistency** - once you assign a name or placeholder to a voice, use the same one throughout

This bracketed format makes it easy to search and replace with actual names later if needed.

**Examples:**
- Known speaker: "[Tim Gallaudet] explains that..."
- Unknown speaker: "[speaker01] explains that..."
- In quotes: > "This is unprecedented" - [Tim Gallaudet]
